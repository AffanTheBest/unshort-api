# Unshort API
An api by which you can extract full links from shortened links.  

---------------------------------------------------------------  

### How to use : 
```
https://unshort.herokuapp.com/api/?url={short-url-here}
```
Example : https://unshort.herokuapp.com/api/?url=https://cutt.ly/qzXU9A2
